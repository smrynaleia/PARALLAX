# PARALLAX

Parallax is a bare‑bones AL project intended for Microsoft Dynamics 365 Business Central.
It serves as a clean starting point for building new extensions.

## Project Overview

This repository contains only the minimal files required for an AL workspace,
including a `.gitignore` tailored for Business Central development. Use it as a
template for your own projects or as a demonstration of how to structure an AL
extension under version control.

## Usage

1. Clone this repository.
2. Open the folder in Visual Studio Code with the AL Language extension
   installed.
3. Configure your environment in `launch.json` and modify `app.json` as needed.
4. Press `F5` to build and deploy the extension to your sandbox.

## Contributing

Contributions are welcome! To propose a change:

4. Describe your changes clearly in the pull request description.
