# Kahrimanlar Danışmanlık Reklam ve Sanayi Ltd. Şti. Web Sitesi

Bu repo, Kahrimanlar Danışmanlık Reklam ve Sanayi Ltd. Şti. için hazırlanan statik ve duyarlı web sitesini içerir.

## Dosya Yapısı
- `index.html`: Ana sayfa
- `about.html`: Hakkımızda
- `services.html`: Hizmetler
- `contact.html`: İletişim sayfası
- `css/style.css`: Stil dosyası
- `js/main.js`: JavaScript fonksiyonları

## Hosting Rehberi
Bu siteyi bir hosting hizmetine yüklemek için aşağıdaki adımları izleyebilirsiniz:

1. **Alan adı ve hosting sağlayıcısı seçin.** Türkiye'de veya globalde hizmet veren herhangi bir sağlayıcı kullanılabilir.
2. **Sunucuya bağlanın.** Sağlayıcınızın sunduğu cPanel, Plesk veya FTP bilgilerini kullanarak bağlanın.
3. **Dosyaları yükleyin.** Bu repodaki tüm dosya ve klasörleri `public_html` (veya kök) dizinine yükleyin.
4. **Ana sayfayı ayarlayın.** `index.html` genellikle varsayılan ana sayfa olarak tanınır; ekstra ayar gerekmeyebilir.
5. **Test edin.** Alan adınızı tarayıcıda açarak sitenin doğru çalıştığını kontrol edin.
6. **Opsiyonel: GitHub Pages veya Netlify.** Eğer klasik hosting yerine ücretsiz bir çözüm isterseniz, 
   - GitHub'da yeni bir depo oluşturup bu dosyaları yükleyin,
   - Depo ayarlarından "Pages" bölümünü etkinleştirin,
   - veya Netlify hesabı açıp repoyu bağlayarak otomatik dağıtım yapın.

Herhangi bir CDN veya cache ayarından sonra değişiklikler biraz gecikmeli yansıyabilir.
